# Hebrew Style Guide

## Course Vocabulary — Use These Terms

### Financial metrics (as students learned them):
- תזרים מזומנים חופשי (FCF) — not just "תזרים"
- יחס מחיר לרווח (P/E)
- תשואה על ההון המושקע (ROIC)
- תשואה על ההון העצמי (ROE)
- רווח למניה (EPS)
- יחס חוב להון עצמי
- יחס שוטף / יחס מהיר
- שולי רווח גולמי / תפעולי / נקי
- הון חוזר (working capital)
- השקעות הוניות (CapEx)
- הכנסות שנתיות חוזרות (ARR)
- שימור הכנסות נטו (NRR)
- עלות גיוס לקוח (CAC)
- ערך חיי לקוח (LTV)

### Investing concepts:
- יתרון תחרותי / חפיר כלכלי (moat)
- עלויות מעבר (switching costs)
- יתרון לגודל (economies of scale)
- אפקט רשת (network effects)
- מונופול / אוליגופול
- מרווח ביטחון (margin of safety)
- יהלום לא מלוטש (turnaround / diamond in the rough)
- בייבק (buyback)
- הקצאת הון (capital allocation)
- שווי פנימי (intrinsic value)
- מכפיל (multiple) — P/E, EV/EBITDA, P/S, P/B

### Hebrew terminology for game UI:
- קנייה (buy)
- העברה / דילוג (pass)
- השפעה חיובית / שלילית / ניטרלית (impact)
- נימוקים (reasoning)
- רמז (hint)

## Tone Guidelines

### DO:
- Conversational but not childish — like a knowledgeable friend explaining
- Use "בדקו" / "שימו לב" / "חשבו על" in hints
- Connect to real-life analogies when possible (from course: pizza slices, house repairs, supermarket vs. corner store)
- Feedback should feel like a mentor walking you through their thinking
- Worked examples: "שלב 1: ... שלב 2: ..." format, show the THINKING not just the conclusion

### DON'T:
- Don't use "מצוין!" or "מעולה!" at the start of every correct answer — vary: "זיהוי נכון", "חשיבה טובה", "הבנת את העיקרון", just start with the explanation directly
- Don't start every incorrect with "טעות!" — vary: "החמצה", "מלכודת ערך קלאסית", just start explaining what happened
- Don't lecture — the player already took the course. Reference principles, don't re-teach them
- Don't use "באפט אמר..." more than once per 5 questions — overused
- Don't use English terms without the Hebrew alongside (always: "FCF (תזרים מזומנים חופשי)" on first use in a question)
- Don't make every company description start with the same structure

### Company Name Style:
- Hebrew company names that sound Israeli-plausible
- 3-5 letter stock symbols that could be TASE tickers
- Mix naming styles: "חברה בע\"מ", "קבוצת חברה", "רשת חברה", just "חברה"
- Sectors should reflect real Israeli economy

### Description Variety:
Start descriptions differently each time. Avoid template feel.
- "רשת של 45 סניפים ב..." (start with size)
- "הוקמה ב-2005 על ידי..." (start with history)
- "מובילת שוק בתחום..." (start with position)
- "אחת מיצרניות ה... הגדולות" (start with category)
- "חברה שקטה שרוב המשקיעים לא מכירים..." (start with narrative)
